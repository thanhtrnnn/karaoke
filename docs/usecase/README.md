# Use Case - Tổng Quan Hệ Thống

## Tổng Quan

## Các Actor
- Thành viên (Member)
- Khách hàng (Client)
- Nhân viên lễ tân (Receptionist)
- Nhân viên phục vụ (Service Staff)
- Quản lý chi nhánh (Branch Manager)
- Chủ doanh nghiệp (Admin)
