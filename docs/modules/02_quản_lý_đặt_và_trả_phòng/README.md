# Module 2: Quản lý đặt và trả phòng

## Tổng quan
Module này quản lý toàn bộ vòng đời đặt phòng: từ khi khách hàng đặt phòng (online/walk-in), check-in, đến trả phòng và thanh toán.

## Use Case chính
- **UC05: Đặt phòng** – Đặt phòng trực tuyến hoặc tại chi nhánh (walk-in)
- **UC07: Quản lý đặt phòng** – Xếp phòng walk-in, xác nhận check-in, tìm kiếm booking
- **UC08: Quản lý trả phòng** – Tính tiền phòng, tổng hợp order, thanh toán, in hóa đơn

## Các Actor liên quan
- **Khách hàng**: Đặt phòng online
- **Nhân viên lễ tân**: Quản lý đặt phòng, quản lý trả phòng

## Mục tiêu chính
1. Cho phép khách hàng đặt phòng trực tuyến
2. Hỗ trợ lễ tân quản lý booking walk-in
3. Quản lý toàn bộ quy trình thanh toán
