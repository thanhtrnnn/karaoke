# PROMPT: Thiết kế giao diện chi tiết hệ thống Quản lý Chuỗi Nhà hàng Karaoke

## Context
Thiết kế toàn bộ giao diện người dùng cho hệ thống quản lý chuỗi nhà hàng karaoke đa chi nhánh. Hệ thống phục vụ 5 nhóm người dùng: Khách hàng, Nhân viên lễ tân, Nhân viên phục vụ, Quản lý chi nhánh, Chủ doanh nghiệp (Admin).

**Tech stack frontend:** React.js (Web), React Native (Mobile cho nhân viên phục vụ).
**Phong cách:** Hiện đại, sạch sẽ, dễ sử dụng. Màu chủ đạo tối (dark theme) với accent màu tím/hồng neon (phong cách karaoke).

---

## CẤU TRÚC TỔNG QUAT

### Layout chung (trừ màn hình Login)
```
┌─────────────────────────────────────────────────────────────┐
│ [Logo]  Tên trang hiện tại          [Chi nhánh] [Avatar ▼] │

│          │                                                  │
│ Sidebar  │              Nội dung chính                      │
│ (menu)   │                                                  │
│          │                                                  │
│          │                                                  │
│          │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

- **Sidebar trái:** Menu điều hướng theo role, icon + text, thu gọn được (collapsed chỉ hiện icon).
- **Header trên:** Tên trang, dropdown chọn chi nhánh (với Admin/Quản lý), avatar + dropdown tài khoản.
- **Nội dung chính:** Thay đổi theo từng trang.
